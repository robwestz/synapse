| Term | Definition | Alias |
|------|------------|-------|
| **SERP** | Search Engine Results Page - Googles sokresultatsida | Sokresultat |

| **Entity** | Namngiven entitet (person, foretag, plats, produkt) | Entitet |
| **BERT** | Bidirectional Encoder Representations from Transformers - sprakmodell for intent-klassificering | - |

| **Word2Vec** | Neural network-modell for ordembeddings | - |
